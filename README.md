#### Front 

-   react
-   react-router
-   react-icons

-   react-redux
-   redux-thunk
-   styled-components
-   sass

#### Back 

-   MongoDB
-   Express
-   Node Js
-   jsonwebtoken
-   nodemailer
-   express-validator

 ## ¿ Cómo levantar el Back?
 
 #### Abrir la terminal
 #### Posicionarse sobre la carpeta en donde se ha clonado el repo
 #### Posicionarse sobre la carpeta /api
 #### npm start
 
  ## ¿ Cómo levantar el Front ?

 #### Posicionarse sobre la carpeta en donde se ha clonado el repo
 #### Posicionarse sobre la carpeta /client
